→ 경쟁자와의 핵심 차이:
  AI 서비스:      패턴 기반 분석 — 심사관의 심리적 반응을 모른다

  로켓커리어연구소: 합격·불합격을 직접 결정해온 HR담당자 집단

```
박스 1 — 지금도 채용을 결정하는 사람들
이력서를 고치는 전문가가 아니라, 실제 합격·불합격을 결정해온
국내 주요 대기업 계열 현직 HR 실무자로 구성됩니다.

박스 4 — 현직이기 때문에 더 솔직합니다
멘토들은 현직자이기 때문에 이름을 밝히지 않습니다.
그래서 회사 눈치 없이, 실제로 보이는 것을 말할 수 있습니다.
```

## 2. 타겟 페르소나

### 페르소나 1 — 방향 없는 진입자 "나침반이 없다"

```
상황:  졸업 예정 or 사회초년생 1년차 미만
고민:  직무 선택 자체가 불확실
       "이 직무가 5년 뒤에도 있을까?"
       "대기업 vs 스타트업, 뭐가 맞는가?"
니즈:  서류 교정이 아니라 방향 설정
트리거: "내가 선택한 방향이 맞다는 확인"
반응:  직무 가치, 커리어 경로, 산업별 현실 콘텐츠
```

### 페르소나 2 — 막힌 3~5년차 "열심히 했는데 어디쯤인가"

```
상황:  첫 이직 or 두 번째 이직 고민 중
고민:  "이직해야 하나, 더 있어야 하나"
       "내 연봉이 시장 대비 적정한가"
니즈:  시장 위치 객관화 + 이동 방향 판단
트리거: "내 서류가 심사관 눈에 어떻게 보이는가"
반응:  연차별 현실, 이직 타이밍, 연봉 구조 콘텐츠
```

### 페르소나 3 — 결단이 필요한 7~10년차 "이번이 마지막 기회일 수 있다"

```
상황:  관리자 트랙 or 전문가 트랙 기로
고민:  "이 경력이 밖에서 얼마나 먹히는가"
       "대기업 → 중견/스타트업 가면 어떻게 되는가"
니즈:  경력의 시장가치 직접 판단
트리거: "HR담당자 눈에 내가 어떻게 보이는가"
반응:  고연차 이직 현실, 직급별 시장 수요 콘텐츠
```

## 3. 메시지 시스템

### 캐치프레이즈 (확정)

```
[캐치 — 공감과 문제 제기]
이직과 취업,
혼자 준비하면 보이지 않는 것들이 있습니다.

[앵커 — 왜 여기서 보이는가]
회사의 운영과 구성원의 커리어를
설계해온 HR담당자들이 직접 함께합니다.
```

**구조 분석:**
- "혼자" → "함께"의 대비가 감정적 해소를 만든다
- 캐치가 만든 외로움을 앵커가 정확히 해소하는 구조
- HR이 주어로 튀지 않고 자연스럽게 권위를 전달

### 메시지 계층

```
Level 1 — 브랜드 (모든 커뮤니케이션의 기반)
"이직과 취업, 혼자 준비하면 보이지 않는 것들이 있습니다"

Level 2 — 차별화 (왜 여기인가)
"회사의 운영과 구성원의 커리어를 설계해온
 HR담당자들이 직접 함께합니다"

Level 3 — 서비스 (무엇을 해주는가)
"HR멘토그룹이 진행하는 커리어 런칭 세션 (무료)"

Level 4 — 액션 (지금 무엇을 해야 하는가)
"지금 세션 신청하기"
```

### 콘텐츠 메시지 원칙

```
❌ 하지 않는 것:
  "자소서 잘 쓰는 법 5가지"           → AI·코치와 구분 안 됨
  "HR담당자가 보는 자소서"             → HR이 주인공으로 튐
  "이렇게 하면 합격합니다"             → 근거 없는 보장

✅ 하는 것:
  "서류 검토하다 이 표현에서 넘어갑니다 — 이유"
  "이 직무, 지금 채용이 열리는 진짜 이유"
  "3년차가 이직할 때 HR이 실제로 보는 것"
  "연봉협상, 우리가 실제로 결정하는 방식"

→ 시점: HR 내부 → 지원자에게 전달
→ 주어: HR이 아니라 "상황"과 "현상"
```

## 4. 채널 전략

### 채널 역할 정의

| 채널 | 역할 | 타겟 | 익명성 | 우선순위 |
|---|---|---|---|---|
| 유튜브 쇼츠 | 최상단 유입 (인지) | 전 페르소나 | ✅ 브랜드채널 | 1순위 |
| 유튜브 롱폼 | 신뢰 구축 (고려) | 페르소나 2·3 | ✅ | 2순위 |
| 카카오채널 | CRM·전환 관리 | 기존+신규 멘티 | ✅ | 즉시 운영 |
| 커뮤니티 | 존재감·트래픽 | 페르소나 1·2 | ✅ 익명 답변 | 보조 |
| 랜딩페이지 | 전환 허브 | 전 채널 유입 | ✅ | 즉시 운영 |
| 크몽 | 현금흐름 유지 | 명확 니즈 보유자 | — | 유지만 |

**링크드인 제외 이유:** 개인 계정 필수 구조 → 익명성 원칙 위반

### 채널별 역할 상세

#### 유튜브 쇼츠 (핵심 채널)

```
목적:    인지 확산 + 알고리즘 기반 구독자 0에서 시작 가능
형식:    60초 이내 / 자막 중심 / 얼굴 노출 불필요
전략:    쇼츠로 주제 검증 → 반응 좋은 주제를 롱폼으로 확장
제작:    스크립트 작성(영석님) + 편집(외주 or 자체)
페이스:  주 2~3편 (초기 채널 성장 구간)
```

#### 유튜브 롱폼

```
목적:    신뢰 구축 + 서비스 전환
형식:    8~15분 / 구조화된 인사이트 전달
전략:    쇼츠에서 검증된 주제만 롱폼화
CTA:     영상 마지막 "커리어 런칭 세션 (무료) 신청" 링크
페이스:  월 2~4편 (쇼츠 채널 안정 후 시작)
```

#### 카카오채널

```
목적:    리드 수집 + 진단 신청 유도 + 전환 관리
운영:    진단 신청 안내 / 리포트 발송 / 팔로업
구독 유도 방식:
  ① 기존 크몽 멘티 → "자체 채널 오픈 + 무료 세션 안내"
  ② 유튜브 CTA → "카카오채널 구독 시 무료 진단 자료 제공"
  ③ 랜딩페이지 → 진단 폼 제출 후 카카오 구독 안내
```

#### 커뮤니티 (블라인드·잡플래닛·오픈카톡)

```
목적:    초기 트래픽 + 타겟 직접 접촉
방식:    취준·이직 관련 질문에 HR 관점 고퀄 답변
         → 프로필/서명에 카카오채널 링크 고정
주의:    스팸 아님 — 가치 먼저 주기 원칙
페이스:  주 5~10개 답변 (시간 투자 최소화)
```

## 5. 콘텐츠 전략

### 콘텐츠 3대 카테고리

#### Category A — "심사관의 눈" (서류·면접)

```
타겟:    페르소나 1·2
목적:    즉각 공감 + 서비스 전환
예시:
  "서류 검토 중 이 표현 나오면 다음 페이지로 넘깁니다"
  "경력기술서에 수치 없는 항목, HR이 하는 생각"
  "자소서 첫 줄에서 이미 결정이 납니다 — 이유"
  "면접관이 질문하면서 실제로 체크하는 것"
```

#### Category B — "커리어의 흐름" (직무·방향성)

```
타겟:    페르소나 1·2·3 전체
목적:    인지 확산 + 브랜드 권위 구축
예시:
  "이 직무, 지금 채용이 활발한 진짜 이유"
  "3년차 이직과 5년차 이직, HR이 다르게 봅니다"
  "대기업에서 10년 있다 나가는 사람들의 공통점"
  "팀장이 되는 사람과 안 되는 사람 — 심사관이 봤을 때"
  "지금 이 산업, 어떤 사람을 급하게 원하는가"
```

#### Category C — "채용 현장의 기준" (연봉·협상·시장)

```
타겟:    페르소나 2·3
목적:    고관여 전환 (고가 상품 연결)
예시:
  "연봉협상, 우리가 실제로 어떻게 결정하는가"
  "이직 시 전 직장 연봉, HR이 어디까지 확인하는가"
  "동일 스펙, 연봉이 다른 이유 — 채용 현장에서 본 것"
```

### 콘텐츠 제작 원칙

```
① 시점은 항상 "HR 내부 → 지원자에게 전달"
   → HR이 주어가 되면 안 됨. 상황과 현상이 주어

② 불편하지만 유용한 진실
   → AI는 듣기 좋게 최적화. HR담당자는 "이거 걸린다"고 직접 말함

③ 구체적 상황으로만 말하기
   → "수치를 넣으세요" (X)
   → "수치 없는 항목 나오면 HR이 하는 생각" (O)

④ 집단의 목소리
   → "제 경험상" (X)
   → "저희가 수백 건 검토하면서 공통적으로..." (O)
```

### 쇼츠 제작 템플릿

```
[0~5초]  후킹 — 즉각 공감되는 상황 제시
          "이 표현 자소서에 쓰셨나요?"

[5~40초] 핵심 인사이트 — HR 내부 시점으로 전달
          "서류 검토할 때 이 표현이 나오면..."

[40~55초] 현실적 처방 — 구체적 행동 안내
          "이렇게 바꾸면 심사관 눈에..."

[55~60초] CTA — 커리어 런칭 세션 (무료)
          "더 알고 싶으면 링크에서 무료 진단 신청"
```

## 6. 퍼널 설계

### 전체 퍼널 구조

```
[인지]
유튜브 쇼츠 / 롱폼 / 커뮤니티 답변
          ↓
[관심]
카카오채널 구독 / 랜딩페이지 방문
          ↓
[진입]
커리어 런칭 세션 신청 (무료 진단 폼 제출)
          ↓
[전환]
AI 초안 + HR멘토그룹 검수 → 리포트 발송 (48h)
→ 영석님 팔로업 1회 ("리포트 어떻게 보셨어요?")
→ 맞춤 상품 안내
          ↓
[구매]
커리어런칭 서비스 결제
          ↓
[재구매·추천]
밸류체인 상품 이동 + 카카오채널 잔류
```

### 커리어 런칭 세션 (무료 진단) 상세 설계

```
명칭:     커리어 런칭 세션 (무료)
부제:     HR멘토그룹이 직접 진행하는 커리어 진단

진입 방법:
  ① 유튜브 → 랜딩페이지 → 진단 폼
  ② 카카오채널 → 진단 폼
  ③ 커뮤니티 답변 → 카카오채널 → 진단 폼

진단 폼 (9문항):
  Q1. 현재 상태 (재직중/구직중/준비중)
  Q2. 목표 직무
  Q3. 현재 연차
  Q4. 목표 기업 유형
  Q5. 가장 큰 고민
  Q6. 현재 보유 서류
  Q7. 지원 희망 시기
  Q8. 본인이 생각하는 약점
  Q9. 유입 경로

처리 프로세스:
  폼 수신 → AI 초안 자동 생성 (1~2분)
  → HR담당자 검수·수정 (5~10분)
  → 리포트 발송 (카카오채널, 48h 이내)
  → 맞춤 상품 링크 포함

리포트 구조 (세일즈 도구로 설계):
  1. 현재 상황 요약 (공감)
  2. 핵심 위크니스 3가지 (충격)
  3. 이 상태로 지원하면 어떻게 되는가 (긴박감)
  4. 솔루션 상품 1~2개 + 구체 링크 (행동 유도)
  5. "지금 신청하면 48시간 내 착수" (마감 효과)
```

### 팔로업 SOP

```
[타이밍] 리포트 발송 후 24시간 내

[방법] 카카오채널 개별 메시지

[스크립트]
"안녕하세요, 커리어 런칭 세션 리포트 잘 받아보셨나요?
 혹시 리포트 보시면서 가장 와닿으셨던 부분이 있으셨나요?
 궁금하신 점 있으시면 편하게 말씀해 주세요."

[목적] 설득이 아니라 대화 시작 + 전환 트리거 확인

[전환 신호] "경력기술서가 문제인 것 같아요" → 상품 안내
[비전환 신호] 무응답 → 7일 후 1회 추가 메시지만
```

## 7. Phase별 실행 로드맵

### Phase 0 — 즉시 (1~2주)

**목표: 퍼널의 최소 단위 가동**

```
채널:
□ 카카오채널 개설 완료 확인
□ 기존 크몽 멘티 전수 카카오채널 유도 메시지 발송
  (자체 채널 오픈 + 무료 세션 안내)

콘텐츠:
□ 유튜브 채널 개설 (로켓커리어연구소 브랜드채널)
□ 쇼츠 첫 3편 기획·제작
  추천 주제:
    ① "이 표현, 경력기술서에 쓰지 마세요 — HR이 하는 생각"
    ② "이직 준비 혼자 하면 모르는 것 3가지"
    ③ "자소서 첫 줄에서 심사관이 결정하는 순간"

퍼널:
□ 랜딩페이지 캐치프레이즈 업데이트
  → "이직과 취업, 혼자 준비하면 보이지 않는 것들이 있습니다"
□ 진단 폼 최종 점검 (9문항)
□ 리포트 세일즈 구조 재설계 (5단계 구조 적용)
```

### Phase 1 — 1~2개월차

**목표: 쇼츠 채널 페이스 확립 + 진단 10건 달성**

```
채널:
□ 유튜브 쇼츠 주 2~3편 유지
□ 커뮤니티(블라인드·잡플래닛) 주 5~10개 고퀄 답변
□ 카카오채널 구독자 50명 달성

콘텐츠:
□ 쇼츠 반응 데이터 분석 (조회수·저장수·댓글)
□ 반응 상위 주제 → 롱폼 기획 준비
□ Category B 콘텐츠 ("커리어의 흐름") 비중 확대

퍼널:
□ 진단 신청 누적 10건 달성
□ 리포트 발송 + 팔로업 100% 실행
□ 전환율 첫 측정 (진단 신청 → 유료 구매)

점검 기준:
  ✓ 쇼츠 평균 조회수 500+ (3편 기준)
  ✓ 진단 신청 10건
  ✓ 팔로업 응답률 확인
```

### Phase 2 — 3~4개월차

**목표: 유튜브 롱폼 시작 + 진단 30건 + 전환율 검증**

```
채널:
□ 유튜브 롱폼 월 2편 시작 (쇼츠 검증 주제 활용)
□ 쇼츠 주 2편 유지
□ 카카오채널 구독자 200명 목표

콘텐츠:
□ 롱폼 CTA 최적화 (커리어 런칭 세션 링크)
□ Category C ("채용 현장의 기준") 콘텐츠 시작
   → 고연차 타겟 유입 목표

퍼널:
□ 월 진단 신청 30건 달성
□ 전환율 20%+ 검증 (6건 이상 유료 전환)
□ 크몽 외 자체채널 매출 10% 확인

점검 기준:
  ✓ 월 진단 30건
  ✓ 전환율 20%+
  ✓ 어느 채널에서 진단이 왔는가 (소스 추적)
  → 전환율 미달 시: 리포트 구조 재검토 (퍼널 문제)
  → 진단 미달 시: 콘텐츠 주제·채널 재검토 (유입 문제)
```

### Phase 3 — 5~6개월차

**목표: 멘토 온보딩 + 자체 결제 + 채널 자립**

```
전제 조건: Phase 2 전환율 20%+ 검증 완료

채널:
□ 유튜브 구독자 1,000명 달성 목표
□ 롱폼 월 4편으로 확대
□ 쇼츠 주 3편 유지

퍼널:
□ 멘토 1~2명 공식 온보딩
□ 토스페이먼츠 자체 결제 도입
□ 월 진단 50건 / 자체채널 매출 40%
□ 크몽 의존도 50% 이하 달성

점검 기준:
  ✓ 멘토 테스트 케이스 3건 완료 (2건 이상 합격)
  ✓ 자체 결제 전환율 (카카오 링크 대비 개선 여부)
  ✓ 월 매출 300만원+
```

## 9. SOP — 채널별 운영 절차

### SOP 1 — 유튜브 쇼츠 제작

```
[트리거] 주 2~3회 제작 사이클

Step 1. 주제 선정 (월요일, 10분)
  - Category A/B/C 로테이션 기준 선정
  - 이전 주 반응 데이터 참고

Step 2. 스크립트 작성 (30분)
  - 후킹(5초) + 인사이트(35초) + 처방(15초) + CTA(5초)
  - HR 내부 시점 언어 사용 확인

Step 3. 촬영·편집 (30~60분)
  - 얼굴 노출 불필요 (자막 + 화면 구성)
  - 브랜드 컬러 딥그린·다크네이비 적용

Step 4. 업로드 (10분)
  - 제목: 검색어 포함 + 호기심 유발
  - 설명란: 커리어 런칭 세션 링크 필수
  - 해시태그: #이직 #취업 #HR #경력기술서 등

Step 5. 반응 확인 (72시간 후, 5분)
  - 조회수·저장·댓글 기록
  - 1,000뷰 이상 → 롱폼 후보 등록
```

### SOP 2 — 커리어 런칭 세션 처리

```
[트리거] 진단 폼 제출 알림 수신

Step 1. 폼 응답 확인 (2분)
  어드민 /admin/inbox 진입 → 응답 전체 확인

Step 2. AI 초안 검토 및 수정 (5~8분)
  - 위크니스 3개 확인·수정
  - 추천 상품 선택 (상황별 기준표 참고)
  - 영석님 코멘트 1줄 추가

Step 3. 리포트 발송 (2분)
  - 카카오채널 개별 발송
  - 상품 링크 포함
  - 상태 업데이트: pending → sent

Step 4. 팔로업 (발송 후 24시간)
  "리포트 잘 받아보셨나요? 가장 와닿으셨던 부분이..."

Step 5. 전환 처리
  - 전환 시: 케이스 생성 → 작업 착수
  - 비전환 시: 7일 후 1회 추가 메시지
  - 전환 여부 기록 (소스·상품·전환일)

총 소요 시간: 건당 10~15분
발송 약속: 48시간 이내
```

### SOP 3 — 주간 운영 사이클

수요일:  커뮤니티 답변 2~3개 / 진단 처리
목요일:  쇼츠 1편 제작·업로드
금요일:  주간 KPI 점검 (조회수·진단신청·전환)
